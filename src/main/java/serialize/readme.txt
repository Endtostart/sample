=======================
        序列化
=======================
目的:
    java对象运行在jvm的内存中,当需要对对象状态进行持久化或者跨网络传输时，就
需要对对象进行序列化和反序列化。
简单点说，序列化就是将java对象通过一系列的操作。将对象的结构和状态从内存中保
存成特定的文件。
反序列化就是将已序列化的java对象文件解析，读取到内存中创建并恢复该对象的状态

序列化方式：
1.jdk提供的序列化成特的那个字节码或者对象流
2.其他第三方序列化工具
  2.1 json格式序列化
    相关工具： jackson，fastjson，gjson
  2.2 字节码序列化
    Kryo
