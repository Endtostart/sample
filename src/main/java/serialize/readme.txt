=======================
        序列化
=======================
目的:
    java对象运行在jvm的内存中,当需要对对象状态进行持久化或者跨网络传输时，就
需要对对象进行序列化和反序列化。
简单点说，序列化就是将java对象通过一系列的操作。将对象的结构和状态从内存中保
存成特定的文件。
反序列化就是将已序列化的java对象文件解析，读取到内存中创建并恢复该对象的状态

序列化方式：
1.jdk提供的序列化成特的那个字节码或者对象流
2.其他第三方序列化工具
  2.1 json格式序列化
    相关工具： jackson，fastjson，gjson
  2.2 字节码序列化
    Kryo
3. json 包中
    Gson,Jackson序列化反序列化
    通过一个统一标准 JsonSelf,和一个标准化的类型TypeTab
    实现了两种序列化的标准化,无需关心具体序列化的供应商是gson还是jackson
    ps：只涉及
    3.1 泛型序列化反序列化
    3.2 反序列化为map
    3.3 复杂对象嵌套序列化反序列化
